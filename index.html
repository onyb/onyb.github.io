                                   About me

I am Ani, a Software Engineer based in Paris.  I work at [Ledger], on the
[Vault] team, building institutional-grade cryptocurrency self-custody.

I'm deeply interested in programming as a craft; i.e. building products that
people <3, powered by </> that's delightful to read.

My current interests revolve around topics underpinning [Bitcoin] and
[Ethereum].  I also have a passive interest in quantum algorithms, and weather
forecasting, thanks to some stints I did in the past.

To know more about my professional work, check out my [CV].  For a more
human-friendly version, scroll down to the Timeline section.

                                   Projects

1. [LittleBit]

   Bitcoin library from scratch; with a clean, readable, and heavily-commented
   code; using nothing but the Python standard library.


2. [ecPoint-Calibrate]

   Interactive meteorological software for calibration of model outputs,
   post-processing, and conditional-verification, for short and medium range
   weather forecasts.

   Development of this project was sponsored by [ECMWF, Reading].

3. [reobject]

   An ORM layer to track and query objects at runtime using a familiar query
   langauge inspired by Django ORM.

4. [HLoader]

   Data-ingestion framework to manage jobs from CMS data-services into CERN
   Hadoop clusters.

   This project was developed at [CERN, Geneva].
